## Project Map 
> 카카오 맵 API 를 이용하여 좌표 기반 검색 사이트 구현 프로젝트 

- 기간 : 2020년 4월 (약 2주간 진행)
- 기술 : `javascript`, `KAKAO Map API`
- 구현기능 
  - 지도 상에서 검색 시, 검색 결과에 맞는 좌표 띄우기
  - 범위 단위로 블록 생성 및 해당 블록 범위에 맞게 좌표 개수 조정하기
  - 각 좌표별 정보 서버로 ajax 통신 통해서 전송
  - 전국 각 지역별 클러스터 설정
